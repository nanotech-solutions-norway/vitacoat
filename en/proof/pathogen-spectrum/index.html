Pathogen Spectrum

Understand the scope of VitaCoat antimicrobial positioning

This page separates VitaCoat pathogen-spectrum positioning into clear technical categories so evaluators can see where the support is strongest and where claim boundaries still matter.

Bacteria

Antibacterial performance is part of the core evidence structure and aligns with the bactericidal testing framework referenced in the technical material.

Yeast / fungi-related contexts

Yeasticidal support forms part of the technical narrative and should be read alongside the relevant standards references.

Enveloped viruses

Enveloped-virus effectiveness is part of the strongest viral claim area in the current VitaCoat material set.

Where support is strongest

  • Bacteria
  • Yeast / Candida-related hygiene contexts
  • Enveloped-virus claim territory

Where caution is required

  • Non-enveloped viruses
  • Generic “all viruses” language
  • Any claim extending beyond the controlled documentation pack
Pathogen-spectrum communication should always be read together with the supporting documentation and the exact testing context. This page should not be reduced to generic “kills everything” wording.
