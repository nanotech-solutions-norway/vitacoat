Pathogen Spectrum

Understanding the Scope of VitaCoat’s Antimicrobial Performance

This page separates VitaCoat’s pathogen-spectrum story into clear technical categories so buyers and reviewers can distinguish where the strongest support exists and where careful claim boundaries must remain in place.

Open DocumentationRequest Technical Review

Bacteria

Antibacterial performance is part of the core VitaCoat proof model and is aligned to the bactericidal testing framework used in the technical material.

Yeasts / Fungi

Yeasticidal support is included in the technical narrative and should be reviewed together with the associated standards references.

Enveloped Viruses

Enveloped-virus performance is part of the strongest viral-claim territory in the current VitaCoat material.

Where the support is strongest

  • Bacteria
  • Yeasts / Candida-related hygiene contexts
  • Enveloped-virus claim territory

Where caution is required

  • Non-enveloped viruses
  • Universal-virus language
  • Any claim extending beyond the documented standards package
Pathogen-spectrum messaging should always be read with the supporting documentation and the exact test context. Avoid reducing this page to generic “kills everything” wording.
